特定のページにレイアウトを適用したい場合はRoute Groupsを使用します。

- (beta)
- (main)

## Private Folders

ルーティングに関係のないディレクトリにはアンダースコアを付けています。

https://nextjs.org/docs/app/building-your-application/routing/colocation#private-folders

例えば、このようなディレクトリが配置されています。

- _utils
- _components
